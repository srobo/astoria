Command Line InterfaceÔÉÅ

astctl is the provided command line interface to Astoria. It is mainly intended for debugging and inspecting the state of Astoria.

For usage information, please see Command Line Usage.

Every command in astctl is implemented as an individual State Consumer, and should be simple enough to understand without detailed documentation.
